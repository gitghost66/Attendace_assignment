# 🤖 AI Resume Screening Agent

Welcome to the **AI-powered Resume Screening System** built with LangGraph and Chainlit.

## How it works

1. 📄 **Upload your Resume** — PDF or DOCX format
2. 📋 **Paste the Job Description** — full text of the JD
3. 🚀 **Watch 9 AI agents analyze the profile** in real-time
4. 📊 **Get a comprehensive hiring recommendation**

## What you'll receive

| Output | Details |
|---|---|
| 🏆 **Hiring Decision** | Strong Hire / Hire / Maybe / Reject |
| 📈 **Overall Score** | Weighted score 0–100 |
| ⚡ **Skill Analysis** | Matched, partial, and missing skills |
| 💼 **Experience Eval** | Years, relevance, career progression |
| 🎓 **Education Eval** | Degree, field, certifications |
| 🤝 **Culture Fit** | Communication, leadership, ownership |
| ❓ **Interview Questions** | 10 personalized, gap-targeted questions |
| 📝 **Full Report** | Complete recruiter-ready document |

## Scoring Weights

- 🛠️ Skill Match — **40%**
- 💼 Experience — **30%**
- 🤝 Culture Fit — **20%**
- 🎓 Education — **10%**
